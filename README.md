# 🎨 AI Placement Recommendation System - Frontend

> 💡 A modern, AI-powered frontend application that enables students to upload resumes, receive personalized placement recommendations, identify skill gaps, and explore learning roadmaps through an intuitive user interface.

---

## 📖 Overview

The **AI Placement Recommendation System - Frontend** is a responsive web application built using modern web technologies. It provides students with an interactive dashboard to upload resumes, view AI-generated job recommendations, analyze skill gaps, track placement progress, and manage their profiles.

---

## ✨ Features

- 🔐 Secure User Authentication
- 🏠 Interactive Dashboard
- 📄 Resume Upload Interface
- 🤖 AI-Powered Job Recommendations
- 📊 Skill Gap Visualization
- 🛣️ Personalized Learning Roadmaps
- 💼 Job Listings & Details
- 👤 User Profile Management
- 📱 Responsive Design
- ⚡ Fast & Modern User Experience

## 📥 Installation

### 1️⃣ Clone the Repository

```bash
git clone https://github.com/Deepakccc/placement-recommendation-frontend.git
```

---

### 2️⃣ Navigate to the Project

```bash
cd placement-recommendation-frontend
```

---

### 3️⃣ Install Dependencies

```bash
pnpm install
```

---

### 4️⃣ Configure Environment Variables

Create a `.env` file in the project root and add the required environment variables.

Example:

```env
VITE_API_URL=http://localhost:3000
```

---

### 5️⃣ Start the Development Server

```bash
pnpm dev
```

---

### 6️⃣ Open the Application

Open your browser and visit:

```
http://localhost:5173
```
